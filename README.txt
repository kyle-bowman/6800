Folder Structure:

1.  .\models
	Contains the actual models from the GridSearchCV in python.
2.  .\outputs
	Various graphs and charts produced from python related to accuracy and times for modeling techniques
3.  .\conclusion.py
	Contains the GridSearchCV programming in python for all of the techniques.  Additionally, contains the charts
	for the computational resources.  This program is associated to the Model Iterations section.
4.  .\Submission_Final.py
	Contains the accuracy and parameter tuning for each of the specific models.  
	This program is associated to the model method overview
5.  .\mammographic_masses.*.* 
	The original data used to generate the analysis
